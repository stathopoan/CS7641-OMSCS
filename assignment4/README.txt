Description
------------
This code is for assignment 4 of machine learning course. It creates plots and results for the report.


Get the code
------------

Fetch the code from my public github repo: https://github.com/stathopoan/CS7641-OMSCS.git

Prerequisites
-------------
Please use Anaconda for easy creation of environment.

1. Navigate to the parent folder assignment3 and locate environment.yml 
2. Type: conda env create -f environment.yml

You will also need to download hiive.mdptoolbox and openai gym. Unfortunately I could not add them to the environment.yml

Run the code
-------------
To run the code just navigate to the parent folder assignment4.

For the whole assignment just run: run_experiments.py script
